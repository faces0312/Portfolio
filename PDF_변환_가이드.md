# 포트폴리오 PDF 변환 가이드

이 도구를 사용하여 HTML 포트폴리오를 PDF로 변환할 수 있습니다.

## 🚀 빠른 시작

### 방법 1: 자동 실행 (Windows)
1. `convert_to_pdf.bat` 파일을 더블클릭하여 실행
2. 자동으로 필요한 패키지를 설치하고 PDF를 생성합니다

### 방법 2: 수동 실행
1. 터미널/명령 프롬프트를 열고 프로젝트 폴더로 이동
2. 필요한 패키지 설치:
   ```bash
   npm install
   ```
3. PDF 변환 실행:
   ```bash
   npm run convert
   ```

## 📋 요구사항

- Node.js (버전 14 이상)
- 인터넷 연결 (패키지 다운로드용)

## 📁 생성되는 파일

- `서현석_Portfolio.pdf` - 변환된 포트폴리오 PDF 파일

## 🎨 PDF 특징

- A4 크기로 최적화
- 모든 프로젝트 섹션이 포함됨
- 아코디언 섹션이 자동으로 열린 상태로 표시
- 페이지 번호 및 헤더 포함
- 인쇄에 최적화된 스타일 적용

## 🔧 문제 해결

### Node.js가 설치되지 않은 경우
1. [Node.js 공식 웹사이트](https://nodejs.org/)에서 다운로드
2. 설치 후 터미널에서 `node --version` 명령어로 확인

### 패키지 설치 오류
```bash
npm cache clean --force
npm install
```

### PDF 생성 오류
- HTML 파일이 올바른 위치에 있는지 확인
- 브라우저가 차단되지 않았는지 확인
- 충분한 디스크 공간이 있는지 확인

## 📞 지원

문제가 발생하면 다음을 확인해주세요:
1. Node.js 버전 (14 이상)
2. 인터넷 연결 상태
3. 파일 권한
4. 디스크 공간
